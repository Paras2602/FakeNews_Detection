# Fake News Detection System

This project is a Machine Learning–based Fake News Detection application that classifies news as **Fake** or **Real** using NLP techniques.

##  Features
- Text preprocessing and vectorization
- Trained ML model for fake news classification
- Flask-based web application
- Simple and user-friendly interface

## Technologies Used
- Python
- Machine Learning
- NLP (TF-IDF)
- Flask
- Scikit-learn

## Project Structure
- `FakeNews_Detection.ipynb` – Model training and analysis
- `app3.py` – Flask application
- `fake_news_model.pkl` – Trained ML model
- `requirements.txt` – Project dependencies
